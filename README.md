# 🎓 SISPENDI
## Sistem Informasi Pendidikan Digital

### Deskripsi
SISPENDI merupakan aplikasi berbasis FlutterFlow dan Firebase yang bertujuan membantu digitalisasi administrasi sekolah, meliputi absensi siswa, absensi guru, monitoring orang tua, dan pembayaran SPP.

## Fitur Utama
- Login
- Dashboard Admin
- Dashboard Guru
- Dashboard Orang Tua
- Data Siswa
- Absensi
- Monitoring Orang Tua
- Pembayaran SPP
- Laporan

## Teknologi
- FlutterFlow
- Firebase Authentication
- Cloud Firestore
- Sheety REST API
- Google Sheets
